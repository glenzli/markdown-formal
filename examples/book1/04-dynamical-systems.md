# 第四章：算子动力系统

:::section {#h-5fd53fffe99f0371 title="演化系统定义"}
:::

在掌握了 @h-fbf90f2b5c5785fa 之后，我们可以正式将算子网络视作一个离散时间动力系统。

:::def {#h-c6fc2320a8d17319 title="演化动力系统"}
给定一个由 @h-15675a1a1a22412b 驱动的网络拓扑，其随时间的离散状态转移方程序列构成了**演化动力系统**。
:::

:::section {#h-ec2081682d4e3e82 title="遍历性分析"}
:::

根据 @h-5815f80196c52eef 所预言的收敛性，以及 @h-47e2c8183f644ed6 提供的边界条件，我们可以证明该系统具有极强的各态历经性质。

:::theorem {#h-778a695d5794c03d title="遍历性定理"}
任何连通的 @h-c6fc2320a8d17319 均是遍历的（Ergodic）。其时间平均算子在无穷远处必然等价于空间平均。
:::

**证明摘要：**
借助 @h-c931336940d20100.title ，我们能够构造一个完备的正交基，由于谱隙（@h-7f4c9ba2db8e7851 ）的存在，所有高阶扰动项都会随时间呈指数级衰减，这就直接引出了 @h-778a695d5794c03d.title 。

:::cor {#h-a36e3fbcfa0431b2 title="强混合推论"}
由 @h-778a695d5794c03d 可推导，该动力系统不仅稳定（见 @h-d9b90e39083c525d ），而且在相空间中满足强混合（Strong Mixing）条件。
:::
