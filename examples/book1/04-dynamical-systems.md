# 第四章：算子动力系统

:::section {#s4j1k2 title="演化系统定义"}
:::

在掌握了 @2e8a5c 之后，我们可以正式将算子网络视作一个离散时间动力系统。

:::def {#5f3b8a title="演化动力系统"}
给定一个由 @b8c3a1 驱动的网络拓扑，其随时间的离散状态转移方程序列构成了**演化动力系统**。
:::

:::section {#s4m3n4 title="遍历性分析"}
:::

根据 @d5a1e7 所预言的收敛性，以及 @8d1f4b 提供的边界条件，我们可以证明该系统具有极强的各态历经性质。

:::theorem {#1a6d9e title="遍历性定理"}
任何连通的 @5f3b8a 均是遍历的（Ergodic）。其时间平均算子在无穷远处必然等价于空间平均。
:::

**证明摘要：**
借助 @6a9c3d.title ，我们能够构造一个完备的正交基，由于谱隙（@4c7e1f ）的存在，所有高阶扰动项都会随时间呈指数级衰减，这就直接引出了 @1a6d9e.title 。

:::cor {#3b8f2c title="强混合推论"}
由 @1a6d9e 可推导，该动力系统不仅稳定（见 @7b4f9c ），而且在相空间中满足强混合（Strong Mixing）条件。
:::
