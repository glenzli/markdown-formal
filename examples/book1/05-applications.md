# 第五章：在量子计算中的应用

:::section {#h-27025c84bb20734d title="量子逻辑门"}
:::

理论必须能够指导实践。在本章，我们将回顾第一章的基础 @h-597fb34638ecf9a3，并运用前面章节构建的重型武器（特别是 @h-778a695d5794c03d 和 @h-a36e3fbcfa0431b2）来分析量子比特网络的演化。

:::example {#h-52c9820bb3571d15 title="量子逻辑门演化"}
考虑一个由随机哈密顿量驱动的量子逻辑门序列。由于系统的幺正性，其演化矩阵是一个特殊的 @h-15675a1a1a22412b。其 @h-fbf90f2b5c5785fa 将完全分布在复平面的单位圆上。
:::

:::section {#h-69380401896817b2 title="耗散收敛"}
:::

注意到在 @h-52c9820bb3571d15 中，虽然满足幺正性，但如果引入耗散环境，系统状态将发生塌缩，这与 @h-5815f80196c52eef 预测的定态收敛现象如出一辙。

:::theorem {#h-64f2911af07fc588 title="量子态耗散收敛定理"}
在开放量子系统下，量子态的演化完全遵循 @h-a36e3fbcfa0431b2 所规定的强混合行为。其收敛时间上限完全由 @h-c931336940d20100 给出的特征根分布所控制。
:::

> **总结：** 从第一章的 @h-15675a1a1a22412b 开始，跨越了 @h-70c80dc9769686c0 的边界理论，直到证明最终的 @h-64f2911af07fc588，这套算子网络理论展现出了惊人的自洽性！
