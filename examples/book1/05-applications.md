# 第五章：在量子计算中的应用

:::section {#s5p5q6 title="量子逻辑门"}
:::

理论必须能够指导实践。在本章，我们将回顾第一章的基础 @s7f9d1，并运用前面章节构建的重型武器（特别是 @1a6d9e 和 @3b8f2c）来分析量子比特网络的演化。

:::example {#9c4a7e title="量子逻辑门演化"}
考虑一个由随机哈密顿量驱动的量子逻辑门序列。由于系统的幺正性，其演化矩阵是一个特殊的 @b8c3a1。其 @2e8a5c 将完全分布在复平面的单位圆上。
:::

:::section {#s5r7s8 title="耗散收敛"}
:::

注意到在 @9c4a7e 中，虽然满足幺正性，但如果引入耗散环境，系统状态将发生塌缩，这与 @d5a1e7 预测的定态收敛现象如出一辙。

:::theorem {#e2f5b1 title="量子态耗散收敛定理"}
在开放量子系统下，量子态的演化完全遵循 @3b8f2c 所规定的强混合行为。其收敛时间上限完全由 @6a9c3d 给出的特征根分布所控制。
:::

> **总结：** 从第一章的 @b8c3a1 开始，跨越了 @c3e8b4 的边界理论，直到证明最终的 @e2f5b1，这套算子网络理论展现出了惊人的自洽性！
