# 第一章：引言

:::section {#s7f9d1 title="背景与定义"}
:::

在正式进入算子网络理论之前，我们先引入一些基本定义和基础命题。这些基础构建块将为后续的跨章节推导打下坚实的基础。

:::def {#b8c3a1 title="线性算子"}
设 $V$ 和 $W$ 是同一个标量域上的向量空间。映射 $T: V \to W$ 称为**线性算子**，如果对于所有 $u, v \in V$ 和标量 $c$，满足 $T(u + v) = T(u) + T(v)$ 且 $T(cu) = cT(u)$。
:::

根据 @b8c3a1.title 的基础性质，我们可以自然引出以下推论。请注意，定义是不参与主编号递增的。

:::prop {#f4a9d2 title="零元映射属性"}
若 $T$ 是一个线性算子，则必然有 $T(0) = 0$。
:::

我们常常需要组合这些算子：

:::section {#s4e2a8 title="算子性质"}
:::

:::lemma {#e7b2c9 title="算子复合的线性性质"}
如果 $T: U \to V$ 和 $S: V \to W$ 均为线性算子，那么它们的复合 $S \circ T: U \to W$ 依然是一个线性算子。
:::

**证明思路**：
通过 @f4a9d2 和基本的向量空间公理，可轻易证明 $S(T(cu + v)) = cS(T(u)) + S(T(v))$。因此 @e7b2c9.title 成立。

:::example {#a1d8f5 title="导数算子"}
在所有处处可导函数构成的空间中，微分算子 $\frac{d}{dx}$ 是一个典型的线性算子，因为它满足 $\frac{d}{dx}(cf + g) = c\frac{df}{dx} + \frac{dg}{dx}$。
:::
