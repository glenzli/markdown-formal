# 第三章：谱理论初步

:::section {#h-95329136daeee49c title="特征值边界"}
:::

在探讨了 @h-5815f80196c52eef 之后，我们有必要深入研究网络演化的谱特性。回想我们在第一章定义的 @h-15675a1a1a22412b ，它是所有后续分析的基础。

:::def {#h-fbf90f2b5c5785fa title="算子谱"}
对于任意有限维 @h-15675a1a1a22412b $T: V \to V$，其**算子谱** $\sigma(T)$ 定义为使得 $T - \lambda I$ 不可逆的所有复数 $\lambda$ 的集合。
:::

基于这一概念，结合第二章中关于迹的 @h-70c80dc9769686c0 ，我们可以得到关于特征值的重要限制。

:::prop {#h-47e2c8183f644ed6 title="特征值边界"}
如果一个有向算子网络满足 @h-70c80dc9769686c0.title ，那么其相应的演化算子的任何非零特征值 $\lambda \in \sigma(T)$ 必定满足 $|\lambda| < 1$。
:::

:::section {#h-eab7be8acbc0a6c4 title="谱半径与谱隙"}
:::

证明在此略去。读者可以参考经典的矩阵分析理论。为了给下一章的动力系统铺垫，我们补充一个关于谱半径的引理。

:::lemma {#h-c931336940d20100 title="谱半径引理"}
在强连通假设（参见 @h-e50a3be6eb728e83 ）下，演化算子的谱半径严格由网络最大权重的反馈环决定。
:::

:::remark {#h-7f4c9ba2db8e7851 title="谱隙说明"}
@h-47e2c8183f644ed6 实际上隐含了谱隙（Spectral Gap）的存在性，这是控制系统达到定态速度的关键因素。
:::
