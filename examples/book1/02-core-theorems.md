# 第二章：核心网络定理

:::section {#s2a1b3 title="基础回顾"}
:::

在第一章中，我们给出了 @b8c3a1 以及一个有趣的 @a1d8f5。并且我们证明了 @e7b2c9。现在我们将利用这些工具。

:::lemma {#c3e8b4 title="网络边界算子引理"}
对于任何封闭的有向算子网络 $G = (V, E)$，如果边缘算子集合都是 @b8c3a1，那么边界循环算子的迹(Trace)必为零。
:::

这是证明主要定理的关键。我们在 @f4a9d2 中已经提到了零元的映射，现在将其扩展到整个网络上。

:::remark {#9f2d6a title="拓扑限制"}
上述引理仅在强连通网络中绝对成立。如果有孤立节点，则需要特殊处理。
:::

:::section {#s2c4d5 title="主要结论"}
:::

现在我们提出本论文的最核心结论：

:::theorem {#d5a1e7 title="算子网络演化定理"}
如果一个算子网络满足 @c3e8b4.title 中的所有条件，那么该网络的长时间演化状态必收敛于唯一的定态，且收敛速度受限于网络的最大非零特征值。
:::

:::cor {#7b4f9c title="网络稳定性推论"}
根据 @d5a1e7，所有合法的算子网络天然具备结构稳定性。即使引入微小的非线性扰动（只要保持宏观的 @e7b2c9 不被破坏），系统最终依然不会发散。
:::

**总结**：
我们从 @f4a9d2 出发，经过 @e7b2c9 和 @c3e8b4，最终确立了 @d5a1e7。这是一个完美的闭环。如果您试图引用一个不存在的 ID，比如 @not-exist，它将会被标红警告。
