# 第六章：指标局部性

:::section {#h-7096f68fd11ad6ba title="局部指标"}
:::

谱序列比较允许我们把全局指标拆解为局部贡献。这里的“局部”指 @h-c474ad4fff1a09be 给出的下降图表，而不是拓扑空间中的开集直觉。

:::def {#h-0bb2b4dc1d14c3e3 title="局部指标"}
给定下降图表上的椭圆复形族，其局部指标定义为每个图表上的 Euler 类与相应 Todd 类的配对。
:::

:::lemma {#h-fc7d79a78a23b7e4 title="余循环抵消引理"}
若交叠项满足 @h-4c2af40e85162c37，则所有二重交叠上的指标误差在 Čech 复形中成边界。
:::

:::section {#h-5ae4f1029ec8d48e title="局部性结论"}
:::

:::cor {#h-c7c32cd3400bb0d8 title="指标局部性推论"}
在 @h-1bec2cb6f1af5555 的假设下，全局指标等于 @h-0bb2b4dc1d14c3e3 给出的局部指标之和；该等式不依赖下降图表的细化。
:::
