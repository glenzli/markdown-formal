# 第五章：谱序列比较

:::section {#h-cb1bbbbb163cf2f6 title="滤过对象"}
:::

为了比较局部与全局不变量，我们给 @h-2d0964dc07820c6f 中的对象赋予有限递增滤过。

:::def {#h-ff34aaf0dea30bb9 title="相容滤过对象"}
相容滤过对象是对象 $E$ 及其滤过 $F_\bullet E$，使得所有限制函子都严格保持滤过并诱导相容的分次对象。
:::

:::lemma {#h-3e864da44591071b title="第一页退化准则"}
若分次对象满足 @h-fa1056dc63b14c69 的统一相位界，则其关联谱序列在有限页之后稳定。
:::

:::section {#h-597bd056c2de883d title="比较定理"}
:::

:::prop {#h-4c2af40e85162c37 title="谱序列比较命题"}
由 @h-ff34aaf0dea30bb9 产生的局部谱序列，经由 @h-22addd5a8114a69e 粘合后，与全局滤过对象的谱序列在 $E_2$ 页之后同构。
:::

@h-4c2af40e85162c37 是第六章指标局部性的主要输入。
