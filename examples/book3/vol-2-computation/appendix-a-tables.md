# 附录 A：数值表格

:::section {#b3-app2-sA-table title="表格格式"}
:::

这是第二卷自己的附录 A，应当重新显示为 `A.1`，而不是接着第一卷附录 B 变成 `C.1`。

:::example {#b3-app2-ex-table-row title="局部表格行"}
一个局部表格行包含网格尺度 $h$、离散谱截断 $\lambda_h$、边界修正 $\eta_h$ 以及最终指标估计值。
:::

:::lemma {#b3-app2-lem-table-consistency title="表格一致性引理"}
若两个网格序列满足 @b3-lem-grid-convergence，则由 @b3-app2-ex-table-row 产生的指标估计在极限中一致。
:::
