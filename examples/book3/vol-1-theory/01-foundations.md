# 第一章：基础对象

:::section {#h-6e64c9c69abaa8b1 title="对象与局部模型"}
:::

我们考虑带边界的紧流形族，并为每个局部图表指定一个可控的解析模型。

:::def {#h-7c173c76886c87a9 title="受控图表"}
受控图表是三元组 $(U,E,D)$，其中 $U$ 是局部坐标域，$E\to U$ 是向量丛，$D$ 是一阶椭圆算子，并且其主符号在边界附近满足一致估计。
:::

:::lemma {#h-4081e32b094b3342 title="局部参数列引理"}
每个 @h-7c173c76886c87a9 都存在局部参数列 $P$，使得 $DP-I$ 与 $PD-I$ 都是紧算子。
:::

:::section {#h-094162a647858ac0 title="相容性"}
:::

:::prop {#h-38e8ece923b5fda8 title="图表粘合命题"}
若两个受控图表在交叠区域上具有相同主符号类，则它们的局部参数列可在紧扰动意义下粘合。
:::
