# 附录 B：符号与取向约定

:::section {#b3-app1-sB-orientation title="取向约定"}
:::

同一卷内的附录 B 应当显示为 `B.1`，并出现在附录 A 之后。

:::remark {#b3-app1-rem-orientation title="取向选择"}
本文采用外法向优先的边界取向约定。该约定会影响 @b3-thm-duality 中指标配对的符号。
:::

:::example {#b3-app1-ex-sign title="一维符号例子"}
在区间 $[0,1]$ 上，外法向优先约定给出边界取向 $-\{0\}+\{1\}$。
:::
